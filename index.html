1 SUM, AVERAGE, COUNT
▾
What it doesSUM adds numbers, AVERAGE finds the mean, COUNT counts cells with numbers.
Syntax
=SUM(B2:B6) → adds B2 to B6 =AVERAGE(B2:B6) → average of range =COUNT(B2:B6) → count of numbers

Power BI for Finance

No code needed — drag and drop. Learn DAX formulas and dashboard layout used in real finance teams.

1 DAX Formula Calculator
▾
What to learnDAX creates custom measures in Power BI. These are the core finance metrics every dashboard needs.
📊 Live DAX Tester
Equivalent DAX in Power BI
// These are the exact Power BI DAX measures Gross Profit = [Total Revenue] - [Total Cost] Profit Margin = DIVIDE([Gross Profit], [Total Revenue]) * 100 YoY Growth % = DIVIDE([Revenue] - [LY Revenue], [LY Revenue]) * 100
Challenge: Set revenue ₹20,00,000 and cost ₹12,00,000. What is the margin and YoY growth?
2 Dashboard Layout Guide
▾
Standard finance dashboard structure
Row 1 — KPI Cards: Total Revenue | Total Expenses | Net Profit | Margin % Row 2 — Trends + Breakdown: Line chart (monthly revenue) | Donut (expense by category) Row 3 — Detail: Matrix table with dept/month breakdown Top bar — Slicers: Month | Region | Department
💡 Build a sample dashboard with Kaggle financial data and add it to your portfolio — it stands out in interviews.
