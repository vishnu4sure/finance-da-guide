Entry Level

Junior Data Analyst

▾
What you will doCollect, clean, and organize financial data. Create basic reports and dashboards for finance teams. Support senior analysts with data preparation.
Key responsibilitiesPull data from databases using SQL, build Excel reports, create basic Power BI dashboards, maintain data accuracy.
Skills needed
ExcelSQLBasic PythonPower BI
Salary range₹4–7 LPA
How to get this roleLearn Excel + SQL first. Build 1–2 dashboard projects. Apply on LinkedIn, Naukri, or Indeed with "Junior Data Analyst Finance" as your search keyword.
Mid Level

Financial Data Analyst

▾
What you will doAnalyze revenue, expense, and investment trends. Build financial models and provide data-driven insights to business leaders.
Key responsibilitiesFinancial modeling, trend analysis, budget vs actual reporting, KPI dashboards, presenting insights to stakeholders.
Skills needed
SQLPythonPower BIFinancial ModelingExcel Advanced
Salary range₹8–14 LPA
How to get this role2–3 years experience as Junior DA. Strong Python + SQL skills. Build a portfolio with financial dashboards and forecasting projects.
Mid Level

Risk Analyst

▾
What you will doIdentify, measure, and report financial risks. Work with banks, insurance companies, and investment firms to protect assets.
Key responsibilitiesCredit risk modeling, market risk analysis, regulatory reporting, stress testing, building risk scorecards.
Skills needed
StatisticsPython/RRisk ModelsSQL
Salary range₹9–15 LPA
How to get this roleStrong statistics background. FRM certification is a plus. Target banks, NBFCs, and insurance companies.
Senior Level

BI Analyst

▾
What you will doBuild enterprise-level dashboards and data pipelines. Translate complex data into actionable strategy for leadership teams.
Key responsibilitiesDesign BI architecture, build Power BI/Tableau dashboards, manage ETL pipelines, define company-wide KPI frameworks.
Skills needed
Power BI / TableauSQLETLData Warehousing
Salary range₹12–20 LPA
How to get this role3–5 years in data analytics. Strong Power BI + SQL skills. Microsoft PL-300 (Power BI) certification recommended.
Senior Level

Quantitative Analyst

▾
What you will doUse advanced mathematics and algorithms to analyze financial markets, build pricing models, and manage investment portfolios.
Key responsibilitiesAlgorithmic trading models, portfolio optimization, derivatives pricing, statistical arbitrage, machine learning for finance.
Skills needed
PythonRStatisticsMachine LearningLinear Algebra
Salary range₹18–35 LPA
How to get this roleMasters or PhD in Math, Statistics, or Finance. Strong Python + ML skills. Target investment banks and hedge funds.
